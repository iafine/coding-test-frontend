Coding Test Frontend
</h1>

## Description

A Blog NextJS SPA used by Coding Test.

- NextJS
- React
- TypeScript
- AntD
- TailwindCSS
- Umi-Request
- Yarn

## Installation

```bash
# install dependencies
$ yarn i
```

## Running the app

```bash
# development
$ yarn dev

# production mode
$ yarn prod
```
